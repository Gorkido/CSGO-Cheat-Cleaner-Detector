# How to build? #
1-) First clone the repository.

2-) Open NuGet Package Manager.

3-) Click "Restore NuGet Packages".

4-) Update the packages.

5-) Restart Visual Studio.

6-) Build it. (Release Any CPU)

7-) And you are ready to go!

8-) Have Fun.
